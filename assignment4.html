3D Chess Set

Goal: Create a 3D chess set using 3D modelling
Time: 2 weeks

Requirements:

  • Model 3D figures using LatheGeometry
  • Create knight figure using ExcludeGeometry
  • Cut the bishop's top using binary operations
  • Save figures' positions using JSON and set them accordingly in the scene
  • Implement spotlight/directional light with interactive parameters and shadows

    • Solution:

  • Models were created using this JSON file
  • Textures were applied to tabletop, chess board and all pieces except for the knights
  • Bishop was cut by creating a rotated cube and subtracting it from bishop's mesh
  • Implemented an interactive shadow and spotlight GUI

    • Notes:

    Currently renderer.shadowMap.type is not set to THREE.PCFSoftShadowMap for better performance.
